# 🔬 PubMed Research Demo

This is a demonstration branch showcasing the PubMed research capabilities. **DO NOT MERGE** this branch into main.

## 🚀 Demo Applications

### Enhanced PubMed Research Hub
```bash
streamlit run pubmed_app.py --server.port=8504
```

### Basic PubMed Search
```bash
streamlit run pubmed_search.py --server.port=8503
```

### Quick Launch Scripts
```bash
# Enhanced PubMed Hub
./run_pubmed_app.sh

# Basic PubMed Search
./run_pubmed.sh
```

## 📋 Features Demonstrated

- **Enhanced Metadata Parsing** - 12+ fields extracted
- **AI-Powered Q&A** - Ask questions about research papers
- **Advanced Analytics** - Keyword frequency, trends, visualizations
- **Professional UI** - Scrollable text, responsive design
- **Export Capabilities** - CSV, JSON, analytics export

## 🔧 Requirements

pip install streamlit pandas plotly langchain-community xmltodict ollama
